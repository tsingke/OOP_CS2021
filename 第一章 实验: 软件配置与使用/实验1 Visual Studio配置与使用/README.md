## VisualStudio快捷


   ```  
   * 单步调试(走马观花F10, 步步深入F11,配合使用)        : 逐行定位BUG
   
   * 设置断点(F9), 直接运行到光标所在代码行(ctrl + F10) : 部分代码调试
   
   * 注释(ctrl+ K + C), 取消注释(ctrl+ K + U)          : 优雅的写代码
   
   * 代码格式化(先选中代码，然后ctrl + K + F)           : 写优雅的代码
   
   * 定义跳转(F12), 跳回原位(ctrl + 减号)               : 查看源码好帮手
   
   * 一键编译调试运行(F5)                               : 简单粗暴又高效                      
   
   ```
